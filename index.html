A toolkit, not another store.

Drop your games on an external drive, map them to Wine prefixes, and launch with the exact GPTK flags each title needs.

Update safeguards
Every toolkit update snapshots your prefix tree. One-click rollback if a runner breaks your save.
Mod stack manager
ModEngine 2, Seamless Coop, Randomizer — toggle layers, see the resolved load order.
Resolved commands

Drive mappings
Map S: X: I: to your external root. Your prefixes stay portable across machines.
Per-game profiles
winver, DLL overrides, MetalFX, AVX, esync — saved per game, never shared.

One install script.

Drop the toolkit on any Apple Silicon Mac running Sonoma or later. The installer wires up GPTK, Wine, and your prefix tree.

install.zsh
$ git clone https://github.com/MoonTheRipper/RipperMoonKit.git
$ cd RipperMoonKit && ./install.zsh
→ Fetching gptk-dsound-nocap-20260513…
✓ Installed to ~/GPTK
✓ Mapped S: X: I: drives
→ Open the app to add games.
