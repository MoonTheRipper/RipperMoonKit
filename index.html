RipperMoonKit

A focused macOS launcher kit for Apple Game Porting Toolkit workflows: profile-based Windows game launches, Steam support, external drive mappings, rollback-safe updates, and the tested Elden Ring ERSC setup.

Download DMG Source ZIP
  • Use your own Apple Game Porting Toolkit 3 download from Apple.
  • Keep Wine prefixes, saves, Steam data, installed games, and GPTK runtimes outside git.
  • Start with the built-in Elden Ring ERSC profile or add individual profiles for other apps and games.
Documentation and source: github.com/MoonTheRipper/RipperMoonKit
