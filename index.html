RipperMoonKit

A clean macOS launcher for Apple Game Porting Toolkit game profiles, copied Windows game folders, Steam support, external drives, and rollback-safe updates.

Download DMG Source ZIP

Proof Of Concept

The screenshots below show the current launcher profile and Elden Ring running on macOS through Apple Game Porting Toolkit 3. The example uses an unofficial copied Windows game folder, Steam support, and an ERSC profile.

What It Helps With

  • Create separate launch profiles for each game or app.
  • Use your own Apple Game Porting Toolkit 3 download from Apple.
  • Map external drives without editing hidden config files by hand.
  • Start or stop Steam separately from the game you are running.
  • Keep saves, prefixes, Steam data, games, and GPTK runtimes outside the repository.
