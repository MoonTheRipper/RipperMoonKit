Installation

Set up the toolkit once.

The installer prepares folders, helper commands, backups, and the local GPTK runtime path. It does not include Apple runtime files or game files.

1. Download GPTK 3

Download Game Porting Toolkit 3 from Apple and mount the downloaded disk image. Leave it mounted while installing RipperMoonKit.

Open Apple Game Porting Toolkit page

2. Install RipperMoonKit

From the source folder, run the installer. It creates local configuration, installs the helper commands, and writes a timestamped log.

cd RipperMoonToolKit
./install.zsh

3. Install The App

The GUI is the easiest path for non-technical users. It installs to your user Applications folder.

zsh scripts/install-gui-app.zsh

4. Reload The Shell

Open a new Terminal window or reload your shell configuration.

source ~/.zshrc

5. Check The Main Commands

which gptk-launch
which gptk-steam
which gptk-game

What Gets Protected

Updates back up the small toolkit files first. Saves, Wine prefixes, Steam data, installed games, GPTK runtimes, and patched runners stay outside the repository and are not removed by normal updates.
