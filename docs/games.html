Games

Use installed game folders, not installer folders.

For many Windows games under GPTK, copying an already-installed game folder from Windows is more reliable than running the original installer.

Folder Shape

Place copied games under your configured games folder. The default external layout looks like this:

/Volumes/GameCoreApp/Games/GameName/
  GameLauncher.exe
  GameData/
  other game files...

Prefixes

A prefix is the Windows-like home for a game or app. Use separate prefixes when games need different dependencies. Use the Steam prefix when the game must talk to the running Steam client.

Saves

Let the game create one new save first. That proves the real save folder. After that, copy backed-up saves into the matching prefix folder and relaunch the game.

External Drives

RipperMoonKit can map as many mounted drives as you need. Assign letters in Settings, keep C: reserved for the Wine prefix, and keep the game paths stable.
