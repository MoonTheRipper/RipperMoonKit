Recommended Reading Order

  1. Start with Installation.
  2. Open Launcher Guide to understand profiles.
  3. Use Games And Saves before copying or moving game folders.
  4. Check Known Tested Games for profile-specific notes before changing launch flags.
  5. Keep Commands nearby for Terminal use.
  6. Use Troubleshooting and Q&A when something behaves differently.
